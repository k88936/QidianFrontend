# React + Vite
# 导师搜索应用

这是一个用于搜索导师信息的React应用。

## 安装说明

1. 克隆仓库
2. 安装依赖: `npm install`
3. 启动开发服务器: `npm run dev`

## 新功能：打字机效果

导师简介部分现在使用打字机效果展示，通过react-type-animation库实现。

## 依赖库

- React 18.2.0
- Material UI 6.4.7
- React Router DOM 7.3.0
- React Type Animation 3.2.0

## 开发命令

- `npm run dev`: 启动开发服务器
- `npm run build`: 构建生产版本
- `npm run preview`: 预览生产版本
